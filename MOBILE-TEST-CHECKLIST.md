# FitBrand Mobile Test Checklist

Test these before launch:

## iPhone Safari
- Home page loads
- Menu opens/closes
- Product cards align correctly
- Cart drawer/page works
- Checkout page is readable
- Profile modal fits screen
- Generator modal scrolls correctly
- My Orders loads

## Android Chrome
- Same checklist as iPhone
- Test keyboard input fields
- Test dropdowns/selects
- Test add-to-cart

## Real checkout test
- Stripe test card works
- User returns to confirmation page
- Order appears in Supabase
- Product access appears on another device after login
