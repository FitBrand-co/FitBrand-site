# FitBrand SEO, Speed & Legal Checklist

## Speed
- Compress JPG/PNG files before launch
- Use WebP/AVIF for new images
- Keep videos short and hosted properly
- Test Lighthouse performance

## SEO
- Each page should have a unique title
- Each page should have a clear meta description
- Add product keywords naturally
- Add Open Graph images for sharing
- Submit sitemap.xml in Google Search Console

## Legal
- Refund policy
- Terms & conditions
- Privacy policy
- Cookie/consent notice
- Contact email
- Clear note that training/nutrition content is guidance, not medical advice

## Payments
- Use Stripe test mode first
- Check webhook events
- Check order/access creation
- Then switch to live mode
